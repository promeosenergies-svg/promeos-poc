# Go-to-Market & Sales — Expert PROMEOS

## 1. Le Wedge Principal : Audit Gratuit 48h

### Pourquoi ca marche
- Valeur immédiate : score + risques en € + 3 actions
- Zéro friction : PDF facture suffit (pas de connexion Enedis)
- Urgence réelle : deadline 11/10/2026 Audit énergétique obligatoire
- Différenciateur : seul acteur à l'automatiser en 48h

### Séquence d'activation du wedge
```
Prospect → Landing page "Audit DT gratuit"
  → Upload 3 dernières factures PDF
  → PROMEOS calcule : score conformité + risque €
  → Email automatique sous 48h avec rapport PDF
  → Appel SDR : "Avez-vous reçu votre rapport ?"
  → Démo live sur les données du prospect
  → Proposition pilote 3 mois
```

### Contenu du rapport d'audit gratuit
1. Score conformité DT : X/100
2. Score BACS : X/100 (si applicable)
3. Risque financier : X € d'amendes potentielles
4. Top 3 anomalies factures détectées
5. Économies estimées : X €/mois
6. Prochaine étape : "Démo complète avec vos données réelles"
