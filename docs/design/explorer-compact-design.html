Key Design Decisions

  • Scope banner → Badge: The full-width blue banner ("Vous explorez Siege HELIOS Paris") becomes a compact <Badge status="info"> inline with the title. Same info, ~40px saved.
  • Filter bar 5→2 rows: Primary filters (site, energy, period, granularity) on Row A. Secondary controls (mode, unit, YoY, actions) on Row B. Everything else in a collapsible "Plus de filtres" section.
  • Actions: label → icon-only: Save/Reset/Copy become icon buttons with tooltips. Saves horizontal space and reduces row height. Full labels still accessible via "Plus de filtres" or tooltips.
  • KPI grid → strip: Follows the existing OverviewRow pattern (flex flex-wrap gap-x-5). Removes icon containers, section title, card borders. Evidence buttons are small inline icons. Confidence badge is inline at end.
  • InfoTip fix: Mode pill tooltips use native title attribute instead of nested InfoTip button. Fixes nested-interactive a11y violation.
  • Subtitle removed: "Visualisez et comparez les courbes..." adds no information — the page title is self-explanatory.
  • Portfolio banner → "Plus de filtres": Only shown when Portfolio mode is active AND user expands the section. Stops consuming vertical space for a rarely-active feature.
  • No design system changes: All components use existing Tailwind classes, Badge component, TrustBadge, and pill patterns already established in the codebase.
