Option B — Rationale

  • Standard disclosure pattern — centered pill on a divider line. Universally understood (HTML details, accordions).
  • Clear visual separation between primary filters (above) and secondary controls (below).
  • Zero learning curve — the "Plus de filtres" label is explicit and self-explanatory.
  • Cost: +18px for the divider row. Doesn't reveal what is hidden until clicked.
  • Best for: Products with many user skill levels, where convention matters more than density.
