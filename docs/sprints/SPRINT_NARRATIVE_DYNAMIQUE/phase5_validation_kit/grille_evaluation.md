# Grille d'évaluation — Critères de réussite Phase 5

Synthèse animateur post-session. Cocher chaque critère **VALIDÉ / RATÉ** + commentaire.

---

## Critère C1 — Lecture 3 min comprise

> Le panel a identifié, pendant les 3 min ou dans les 30 sec post-test (Q ouverte), au moins **4 critères sur 5** de la grille typologie ci-dessous.

### Grand groupe tertiaire (CFO / Asset Manager)
- [ ] Comprendre le **niveau d'exposition financière** (€)
- [ ] Comprendre le **score de conformité** (X/100 ou trajectoire 2030)
- [ ] Identifier **au moins 1 site / portefeuille en dérive**
- [ ] Comprendre l'**action prioritaire** de la semaine
- [ ] Identifier au moins **1 levier d'économies** (MWh/an ou €)

### Commerce (boulanger / pharmacien)
- [ ] Comprendre le **surcoût ou écart vs voisins similaires** (€/mois)
- [ ] Identifier **où se situe le problème** (consommation hors heures, etc.)
- [ ] Comprendre l'**action concrète à prendre** cette semaine
- [ ] Identifier **au moins 1 levier d'économies** chiffré
- [ ] Repérer **1 anomalie** (facturation, dérive)

### ERP (directeur école / EHPAD)
- [ ] Comprendre l'**état de conformité réglementaire** (OPERAT / DT)
- [ ] Identifier **l'échéance la plus proche** (déclaration / audit)
- [ ] Comprendre l'**impact budget** ou **service public**
- [ ] Identifier **au moins 1 levier d'économies** mobilisable
- [ ] Repérer **1 alerte** ou **1 action prioritaire**
