5 modes stratégiques — référence visuelle

La page Synthèse stratégique est polymorphique. Le moteur d'assujettissement (ADR-024) calcule un mode unique parmi cinq, et chaque mode produit une narration, un triptyque KPI et une paire de charts spécifiques. Le frontend ne fait jamais le tri : il consomme payload.strategic_mode et dispatche.

Cascade de gates (compute_strategic_mode)

Gate 1 · DATA_INSUFFICIENT   if maturity < 0.60 OR unknown_ratio > 0.30 → DATA_INSUFFICIENT
Gate 2 · REGULATORY_DRIVEN   if (DT or BACS applicable) AND trajectory_drift > 5 % → REGULATORY_DRIVEN
Gate 3 · PROCUREMENT_DRIVEN   if next_contract_end < 90j OR spot_exposure > 40 % → PROCUREMENT_DRIVEN
Gate 4 · OPPORTUNITY_DRIVEN   if APER applicable OR unvalued_cee > 50 k€ → OPPORTUNITY_DRIVEN
Default · PERFORMANCE_DRIVEN   sinon → PERFORMANCE_DRIVEN
