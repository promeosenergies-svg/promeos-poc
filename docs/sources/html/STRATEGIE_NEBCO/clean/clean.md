## RÉSUMÉ EXÉCUTIF 

NEBCO (Notification d’Échange de Bloc de Consommation) est le mécanisme réglementaire français permettant aux consommateurs d’électricité de valoriser sur les marchés énergétiques leurs réductions de consommation ( effacement ) ET les décalages avant/après ( reports/anticipations ). Depuis le 1er septembre 2025, il remplace le système NEBEF. 1 2 

Pour PROMEOS , NEBCO offre un axe de croissance crédible mais exigeant . Le marché s’élargit : flexibilité française de 2,9 GW (2024, +8% YoY) vers 6,5 GW en 2028. Les segments cibles (retail multisites, PME/ETI tertiaire, petits industriels) sont accessibles via la plateforme advizeo existante. Or, l’exécution requiert partenariat OE structuré, qualification RTE coûteuse, et données fiables (GTB/télérelève). 3 

Verdict provisoire : NEBCO n’est pas un MVP seul , mais un module V1 viable s’il est bundlé avec audit d’efficacité énergétique et pilotage cockpit. Potentiel : €500k–1M revenue Y1 (50–100 sites), €1.5–3M Y2. 

Recommandation immédiate : Lancer pilot 30 jours (audit pré-flex 2–3 clients, partenariat OE, validation data) pour décision GO/NO-GO conditionnel. Deadline décisions: 22 février 2026 . 

## 1. FAITS CLÉS (SOURCÉS) 

### Définition & Cadre Réglementaire 

NEBCO, défini dans le Code de l’énergie (articles L.271-1, L.271-2, L.321-15-1, R.271-1) et approuvé par la CRE délibération 2025-199 du 23 juillet 2025, permet la valorisation de trois types de modulations de consommation: 4 
- Effacement (réduction) – Modulation à la baisse classique 
- Report – Hausse APRÈS l’effacement (dans les 7 jours Sites Télérelevés ; 2 jours Profilés) 
- Anticipation – Hausse AVANT l’effacement 

L’innovation majeure vs. NEBEF : la valorisation explicite des reports/anticipations , reconnaissant que les sites compensent après arrêt d’équipements. 5 

### Acteurs & Responsabilités Acteur Responsabilité Clé RTE Certifie effacements ; délivre Agréments Techniques ; contrôle M&V ; publie données Opérateur d’Effacement (OE) Signe Accord de Participation ; déclare Programmes Modulation ; verse Fournisseurs Gestionnaire Réseau Distribution (GRD) Fournit courbes charge (15 min ≤36kVA ; 5 min >36kVA) ; conventions techniques Site Soutirage Donne accord écrit ; réalise modulation dans contraintes métier 

### Marché & Dynamique 2024–2025 
- Capacité totale (2024) : 2,9 GW (+8% YoY, 4e augmentation consécutive) 6 
- Objectif 2028 : 6,5 GW 7 
- Gisement géographique : Anciennement industrie lourde ; nouvelle opportunité en tertiaire (bureaux, retail, hôtellerie) 
- Prices EPEX spot : 72,89 €/MWh (jan 2026) ; pics 122 €/MWh (fév 2025) ; spreads volatiles 8 
- Mécanisme capacité (2025) : 0 €/MW (suroffre) ; refonte prévue 2026 9 

### Éligibilité & Prérequis 

Tous les sites France métropolitaine continentale éligibles si : 
- Accord écrit du titulaire CART/CARD 10 
- Puissance effaçable ≥ 100 kW minimum 
- Data : Télérelevé (PS ≥ seuil profilage) OU Profilé (<seuil) OU sous-mesure expérimentale (jusqu’au 31/12/2025, limites 100 sites/OE) 
- Agrément Technique Modulation Consommation (OE must pass RTE tests ; fiabilité ≥50%) 11 
- Conformité bilan énergétique : Volumes hausse ≤ volumes baisse (7–2 jours glissants) ; bilan mensuel/annuel ≥ -5%, sinon suspension M+2 (capacité réduite 60%) 12

## 5. RISQUES MAJEURS & MITIGATIONS 

### A) Réglementaire 
- Risque : Audit RTE pénalités (€0–100k/an OE si non-conformité M&V) 
- Mitigation : Dashboard alertes temps réel ; monitoring strict baseline ; taux réussite ≥80% 

### B) Opérationnel 
- Risque : Dérives baseline, non-disponibilité sites, bruit données → réalisé invalide 
- Mitigation : Sub-comptage qualifié (IoT) + validation pré-activation ; redundancy data 

### C) Financier 
- Risque : Volatilité spread (€20–200/MWh) ; impossible promettre gains stables 
- Mitigation : Communicate ranges (3 scénarios) ; success-fee sur net spread seulement 

### D) Contractuel 
- Risque : Ambiguïté responsabilité pénalités (OE vs. Client vs. PROMEOS) 
- Mitigation : Clauses cristal-claires : OE = responsable légal M&V ; PROMEOS = audit advisor 

### E) Réputation 
- Risque : Surestimation gains → client déception → brand damage 
- Mitigation : Estimates conservateurs (italicize volatilité) ; 3e-party audit crédibilité 

## 6. PRÉREQUIS & INFRASTRUCTURE 

### Données Minimales 
- Granularité : 10–30 min sub-metering; GRD fournit 15 min (≤36kVA) / 5 min (>36kVA) 
- Historique : 2–3 ans données consommation 
- Télérelève : Active, pas de lacunes 
- Métadonnées : Équipements, P. souscrite, horaires, arrêts prévus 

### Capacité Pilotage 
- GTB/BMS : Contrôle automates ; commande distance 
- IoT : Sous-comptage, capteurs T°, présence (optionnel, +précision) 
- Contraintes métier : Confort min, % production loss toléré 
- Horaires : Fenêtres opérationnelles (7h–20h? Weekends? Vacances?) 

### IT Infrastructure 
- Pipeline données (daily sync) : advizeo SaaS ~€150–300/site/an 
- Moteur règles : Baseline vs. réalisé ; détection écarts 
- Journal audit : Logs RTE-compliance 
- Alerting : Seuils dépassés (email/SMS) 

### Partenaires Must-Have 
- OE opérateur (Energy Pool, Enerdigit) : Contractualisation RTE, M&V cert., risk bearing 
- Advizeo cockpit : EMS/BMS monitoring, data integration 
- Installateur IoT (si needed) : Pose sous-compteurs

## 8. SEGMENTATION CIBLE (ICP) 

### Trois Wedge Use-Cases Réalistes 

### 1. Retail Froid Alimentaire (Supermarchés 3k–5k m²) 
- Flexibilité : Arrêt réfrigérateurs 30–60 min; inertie thermale maintient T° 
- Gisement : 500 kW–1.5 MW ; ~50 jours/an profitable 
- Revenu/site : €8–20k/an (30 sites cluster = €240k–600k potentiel) 

### 2. CVC Tertiaire & Bureaux (PME 1k–20k m²) 
- Flexibilité : Pre-cool 1–2h avant pic ; relâcher 0.5–1°C confort 
- Gisement : 200–800 kW ; ~60 jours/an 
- Revenu/site : €3–10k/an (50 PME = €150–500k potentiel) 

### 3. Process Léger Agroalimentaire (Broyage, Emballage, Dosage) 
- Flexibilité : Arrêt ligne 30–120 min (produit stockable pré/post) 
- Gisement : 500 kW–2 MW ; 20–40 jours/an 
- Revenu/site : €10–40k/an (20–30 PME = €200–1.2M potentiel)

### Décision 1: Partenariat OE — Make vs. Partner 

Recommandation : ✅ PARTNER (Energy Pool ou Enerdigit) 
- Why : CapEx €50–100k (vs. €200–500k in-house) ; timeline 6–8 sem (vs. 12 mois) 
- Action : Contact Energy Pool partnership director J1 ; propose pilot mutualisé 50–100 sites Q2 2026 

### Décision 2: Focus Segment V1 

Recommandation : ✅ RETAIL PRIMARY (Carrefour/Burger King touchpoints); Tertiaire SECONDARY 
- Why : Mature market (DEET/BACS pressure), centralized GTB, existing advizeo clients 
- Action : Pilot audit = Carrefour 5 magasins ; roadmap 20–50 sites Q2–Q3 2026 

### Décision 3: Data Scope Minimal 

Recommandation : ✅ GRD + Strategic Sub-metering (froid + CVC key equipment only) 
- Why : 80% precision at 20% cost ; 2–4 week deployment (vs. 8–12 weeks full IoT) 
- Action : Pilot = GRD (free) + 2–3 sous-compteurs critiques (~€500–1k total setup) 

### Décision 4: Capital & Resource Allocation 

Recommandation : €150–250k pilot investment Y1 (vs. €500k–1M scale) 
- Breakdown: €50k OE partnership setup ; €40k cockpit/IoT pilot config ; €30k legal/compliance ; €20k audit costs (sunk) ; €10k contingency 
- Headcount : 1 FTE Tech Lead (pilot ) + Sales/Ops support 

### Décision 5: Go/No-Go Threshold (J30) 

Recommandation : Advance IF ≥4/5 conditions validated : 
- ✅ OE Partner LOI signed 
- ✅ ≥2 retail LOI prospects (audit interest) 
- ✅ Advizeo cockpit M&V-capable (tech sign-off) 
- ✅ Contrats templates finalized 
- ✅ €1M+ Y1 revenue potential identified 

If 3/5 or fewer → EXTEND J60 ; refine model.

NEBCO is strategically attractive but operationally demanding. PROMEOS can capture €500k–1M Y1 revenue and €1.5–3M Y2 if partnerships are structured (Energy Pool), data infrastructure proven (advizeo cockpit), and execution ruthlessly disciplined (50+ sites scale, compliance-first). 

This is not a quick-win wedge, but a scalable B2B motion — especially bundled with energy efficiency (DEET/BACS), where client ROI is 3x higher and churn lower.
